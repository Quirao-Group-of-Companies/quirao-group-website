# Quirao Group of Companies Website

This repository hosts the source code for the QGC website. Developed by QGC's interns from the Technical Operations and Data Department within February to April of 2026.

## Get Started
To set the development environment:
```bash
git clone <REPOSITORY-URL>
cd quirao-group-website/

bun install # Install necessary dependencies

bun run build # To ensure that you do not have issues in building the application

bun run dev:web # To run the web application
```

For more details, visit the `docs/` directory from root of the project.
